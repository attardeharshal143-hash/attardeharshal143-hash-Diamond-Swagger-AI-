Gmail Integration Setup

Real Gmail Integration Setup Required

Currently using demo mode. Follow these steps to enable real Gmail integration:

📋 Step-by-Step Setup Guide

1. Google Cloud Console Setup

  1. Go to Google Cloud Console
  2. Create a new project or select an existing one
  3. Enable the Gmail API in the API Library
  4. Go to "Credentials" and create OAuth 2.0 Client ID
  5. Set application type to "Web application"
  6. Add your domain to "Authorized JavaScript origins"

2. Update Code Configuration

Replace these values in the code:

// Replace these placeholder values:
const GMAIL_CLIENT_ID = 'your-actual-client-id.googleusercontent.com';
const GMAIL_API_KEY = 'your-actual-api-key';

3. OAuth Consent Screen

  • Configure OAuth consent screen in Google Cloud Console
  • Add your email as a test user (for development)
  • Request Gmail read-only scope: https://www.googleapis.com/auth/gmail.readonly

4. Security Considerations

  • Use environment variables for production deployment
  • Implement proper error handling for API failures
  • Consider rate limiting for Gmail API calls
  • Store OAuth tokens securely
